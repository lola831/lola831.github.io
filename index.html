Cod Mountain Fish Co. In Progress

Currently under active development, Cod Mountain Fish Co. is a dynamic web application tailored for a fishing charter business based in Santa Cruz, CA. As a freelance full stack software engineer, I am responsible for both the front-end and back-end development. The front-end leverages HTML, Tailwind CSS, React, and JavaScript for a seamless user experience, while the back-end utilizes Python with Flask and MongoDB for efficient data handling. The website features a custom-built availability and booking calendar that allows customers to easily view slots and book charters online, complemented by Square for secure payment processing. Additionally, it includes a comprehensive captain's dashboard for managing reservations, blocking days, and updating fish reports and recipes. AWS S3 is used for image hosting, and EmailJS is integrated for sending confirmation emails.
